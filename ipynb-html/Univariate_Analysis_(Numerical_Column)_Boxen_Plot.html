4. How to Interpret Boxen Plot :

Identify the distribution of the data: Look at the y-axis to understand the variable being analyzed.

Compare the distribution of values:

  • The boxen plot is made up of several "boxes" of varying heights. Each of these boxes represents a different portion of the data's distribution.
  • The height of the boxes shows how much of the data falls within that range. Taller boxes mean more data points are in that range; shorter boxes mean fewer data points.
  • The median is shown by a horizontal line within the boxes.
  • The overall shape of the boxen plot gives you a sense of whether the data is symmetrical or skewed.

Look for patterns:

  • Density: Boxen plots are designed to show the density of the data. Where the boxes are taller, the data is more concentrated.
  • Skewness: If the boxes are concentrated towards the bottom of the plot, the data is likely skewed to the right (positive skew). If they're concentrated towards the top, the data is skewed to the left (negative skew).
  • Outliers: Boxen plots can help you spot potential outliers, though they don't explicitly mark them in the same way a standard box plot does. Areas where the boxes become very short or sparse may indicate the presence of outliers.

Consider the context:

  • The meaning of the data depends on what is being measured. For example, if it represents test scores, you can see how the scores are distributed across the class.

Sepal Length Distribution

  • Shape: The median is located slightly towards the bottom of the box, and the upper whisker is slightly longer than the lower whisker. This suggests a slightly right-skewed distribution.
  • Center: The median sepal length is around 5.8 cm.
  • Spread: The box is moderately sized, indicating a moderate spread in the middle 50% of the data. The whiskers show that the overall range of the data is also moderate.
  • Outliers: There are a few points outside the upper whisker, indicating the presence of some outliers with unusually large sepal lengths.

5. Visualize Boxen plots with jitters

A. Create a Boxen plot with jitters

In[15]:
import numpy as np
# Create the boxen plot with jittered data points
plt.figure(figsize=(8, 6))  # Adjust figure size for better readability

# 1. Create the boxen plot
sns.boxenplot(
    y=df["sepal_width"],
    color="red",
    linewidth=1,
    width=0.8,
    k_depth="full" # Added k_depth parameter, 'full' shows a similar look to the original boxen plot
)

# 2. Overlay the jittered data points
#  -  'y' is the column to plot
#  -  'x' is set to 0 to plot points along the boxen plot
#  -  'jitter' adds random horizontal displacement
sns.stripplot(
    y=df["sepal_width"],
    x=np.zeros(len(df)),  # Position at x=0 (center of the boxen)
    jitter=0.2,             # Amount of jitter (adjust as needed)
    color="red",         # Color of the points
    alpha=0.5,             # Transparency of the points
    size=5                 # Size of the points
)

plt.title("Sepal Width Distribution with Jitter")
plt.ylabel("Sepal Width (cm)")
plt.xlabel("")  # Remove the x-axis label since it's not meaningful here

plt.show()

B. Create multiple Boxen plot with jitters

# Define the columns to plot
columns_to_plot = ['sepal_length', 'sepal_width', 'petal_length', 'petal_width']

# Define colors for each boxen plot
colors = ['red', 'green', 'orange', 'coral']

# Create a 2x2 subplot grid with a fixed figure size
fig, axes = plt.subplots(2, 2, figsize=(14, 10))

# Iterate through the columns and plot each one in a separate subplot
for i, col in enumerate(columns_to_plot):
    # Determine the row and column for the current subplot
    row = i // 2
    col_num = i % 2

    # Select the appropriate subplot axis
    ax = axes[row, col_num]

    # Create the boxen plot, using a different color for each
    sns.boxenplot(data=df, y=df[col], color=colors[i], ax=ax, k_depth="full") # added k_depth

    # Add the jittered points
    sns.stripplot(
        y=df[col],
        x=np.zeros(len(df)),  # Position at the center of the boxen
        jitter=0.2,             # Adjust jitter as needed
        color="black",
        alpha=0.5,
        size=5,
        ax=ax
    )

    # Set the title and labels for the subplot
    ax.set_title(f"{col.title()} Distribution with Jitter")
    ax.set_ylabel(f"{col.title()} (cm)")
    ax.set_xlabel("")

# Adjust the layout to prevent overlapping titles/labels
plt.tight_layout()

# Display the plots
plt.show()

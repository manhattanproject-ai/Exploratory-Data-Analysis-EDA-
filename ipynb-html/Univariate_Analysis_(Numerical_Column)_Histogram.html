D. Visualize multiple histograms together

  1. Define the columns to plot ( update based on your scenario )
  2. Define the colors of each histogram ( update based on your choice/preference )
  3. we created a 2 * 2 matrix with each figure as (14 , 10 ) , you can update each of them based on preference

# Define the columns to plot
columns_to_plot = ['sepal_length', 'sepal_width', 'petal_length', 'petal_width']

# Define colors for each histogram
colors = ['red', 'green', 'orange', 'coral']  # Add more colors if needed

# Create a 2x2 subplot grid with a fixed figure size
fig, axes = plt.subplots(2, 2, figsize=(14, 10))

# Iterate through the columns and plot each one in a separate subplot
for i, col in enumerate(columns_to_plot):
    # Determine the row and column for the current subplot
    row = i // 2
    col_num = i % 2

    # Select the appropriate subplot axis
    ax = axes[row, col_num]

    # Create the histogram with KDE, using a different color for each
    sns.histplot(data=df, x=col, color=colors[i], label=col, kde=True, binwidth=0.2, alpha=0.7, stat="density", ax=ax)

    # Set the title and labels for the subplot
    ax.set_title(f"{col.title()} Distribution")
    ax.set_xlabel(f"{col.title()} (cm)")
    ax.set_ylabel("Density")
    ax.legend()

# Adjust the layout to prevent overlapping titles/labels
plt.tight_layout()

# Display the plots
plt.show()

4. How to Interpret Histogram :

Key Features to Look For

Shape:

  • Symmetric: The left and right sides of the histogram are roughly mirror images.
  • Skewed: The histogram has a "tail" extending more to one side.
    • Right-skewed (positive skew): The tail extends to the right.
    • Left-skewed (negative skew): The tail extends to the left.
  • Unimodal: The histogram has one peak.
  • Bimodal: The histogram has two peaks.
  • Multimodal: The histogram has more than two peaks.

Center: Where is the data concentrated? Look for the peak of the histogram, which gives you an idea of the typical value.

Spread: How much does the data vary?

  • Narrow histogram: Data points are clustered close together.
  • Wide histogram: Data points are more spread out.

Outliers: Are there any isolated bars far from the main body of the histogram? These could represent unusual data points.

Gaps: Are there any significant gaps in the data, where there are no bars?

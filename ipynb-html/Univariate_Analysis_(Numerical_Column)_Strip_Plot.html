C. Visualize multiple Strip plot together¶

  1. Define the columns to plot ( update based on your scenario )
  2. Define the colors of each Strip plot ( update based on your choice/preference )
  3. we created a 2 * 2 matrix with each figure as (14 , 10 ) , you can update each of them based on preference
In [14]:
# Define the columns to plot
columns_to_plot = ['sepal_length', 'sepal_width', 'petal_length', 'petal_width']

# Define colors for each strip plot
colors = ['red', 'green', 'orange', 'coral']  # Add more colors if needed
markers = ['o', '*', 's', 'd'] #marker list

# Create a 2x2 subplot grid with a fixed figure size
fig, axes = plt.subplots(2, 2, figsize=(14, 10))

# Iterate through the columns and plot each one in a separate subplot
for i, col in enumerate(columns_to_plot):
    # Determine the row and column for the current subplot
    row = i // 2
    col_num = i % 2

    # Select the appropriate subplot axis
    ax = axes[row, col_num]

    # Create the strip plot, using a different color for each
    sns.stripplot(data=df, y=df[col], color=colors[i], marker=markers[i], size=8, ax=ax) #added marker

    # Set the title and labels for the subplot
    ax.set_title(f"{col.title()} Distribution")
    ax.set_ylabel(f"{col.title()} (cm)")  # Changed from xlabel to ylabel
    ax.set_xlabel("")

# Adjust the layout to prevent overlapping titles/labels
plt.tight_layout()

# Display the plots
plt.show()

4. How to Interpret Strip plot :¶

Key Features to Look For

Shape:

  • A strip plot shows the distribution of individual data points.

  • Since a strip plot displays individual data points, the "shape" you observe is how those points are distributed.

  • Symmetric: If the points are distributed evenly on both sides of the center, the distribution is symmetric.

  • Skewed:

    • Right-skewed (positive skew): Most points are concentrated on the left, with fewer points extending to the right.
    • Left-skewed (negative skew): Most points are concentrated on the right, with fewer points extending to the left.

  • Unimodal/Bimodal/Multimodal: Clusters of points can indicate peaks in the data.

    • Unimodal: A single dense cluster.
    • Bimodal: Two dense clusters.
    • Multimodal: More than two dense clusters.

  • Center: The center of the data can be visually estimated. It's where the points are most concentrated.

  • Spread: The spread is how much the points vary along the axis. A wider spread means the data points have more variability.

  • Outliers: Outliers are easily identifiable as points that are far away from the main cluster of data points.

  • Gaps: Strip plots can show gaps in the data. Gaps appear as empty areas along the axis where there are no points.
